## Что такое CSS-правило?

CSS-правило (CSS rule) — это основная единица кода в языке CSS, которая описывает, как определенные элементы HTML должны быть стилизованы. Каждое CSS-правило состоит из селектора и блока объявления.

### Структура CSS-правила

1. **Селектор (Selector)**: Указывает, к каким HTML-элементам будет применяться стиль. Это может быть тег, класс, идентификатор или комбинация различных селекторов.

2. **Блок объявления (Declaration Block)**: Содержит одно или несколько свойств и их значений, заключенных в фигурные скобки `{}`. Свойства определяют, какие стили будут применены к выбранным элементам.

### Пример CSS-правила

```css
/* Селектор для всех элементов <p> */
p {
    color: blue; /* Свойство color задает цвет текста */
    font-size: 16px; /* Свойство font-size задает размер шрифта */
    line-height: 1.5; /* Свойство line-height задает межстрочный интервал */
}
```

### Как работает CSS-правило

1. **Селектор** `p` выбирает все элементы `<p>` на странице.
2. Блок объявления содержит три свойства:
   - `color: blue;` — задает цвет текста параграфов синий.
   - `font-size: 16px;` — устанавливает размер шрифта для текста параграфов на 16 пикселей.
   - `line-height: 1.5;` — определяет межстрочный интервал, равный 1.5.

### Важные моменты

- **Каскадность**: Если несколько правил применяются к одному и тому же элементу, CSS использует принцип каскадности для определения, какое правило имеет приоритет. Обычно правило, объявленное позже, имеет более высокий приоритет, если у них одинаковая специфичность.

- **Специфичность**: Это система, по которой браузер определяет, какое правило применить к элементу, основываясь на селекторах. Более специфичные селекторы имеют приоритет над менее специфичными.

Таким образом, CSS-правило является ключевым элементом в стилизации веб-страниц и позволяет разработчикам управлять внешним видом элементов на странице.
